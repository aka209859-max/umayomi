## 🆕 追加フォルダ（TARGET独自）

TARGET frontier JVでは、以下の独自フォルダも存在しますが、**JRA-VANの公式データフォルダではありません**:

- **JG_DATA**: 騎手情報（UMAYOMI独自パーサー `jg.ts` で実装済み）
- **FUKU_DATA**: 勝負服データ（表示用、回収率分析に不要）
- **TM_DATA**: 対戦型マイニングデータ（低優先度）
- **W5_DATA**: WIN5データ（低優先度）
- **MY_DATA**: ユーザー入力データ（買い目、コメント等）
- **RT_DATA**: 速報データ（当日のみ使用）
